Primitive Data

  • There are eight primitive data types in Java and they are: boolean , byte , char, short, int,long, float
  • One of the fundamental rule of Java is you can't fool around with primitive data types.
  • But as always there is indeed a way out, Java uses the concept of a Wrapper class for all eight primitive types- In the case of an int we can use Integer, and by doing that it gives us ways to perform operations on an int.
  • Integer minimum value = -2147483648 and Integer maximum value = 2147483647
  • If you add value to the max value computer will overflow and reverse the integer , i.e. if you try to get max+1 you will end up with the min. Same happens for the minimum number and it is known as underflow.
  • Byte Min = -128 and Byte Max = 127. When you use values in this range you should use byte as it makes the code much faster than the general methods.
  • A Byte occupies 8 bits, A short occupies 16 bits, An int occupies 32 bits, Long occupies 64 bit.
  • So for this reason if you can get away with byte don't even bother moving to int.
  • To define an long we have to use 'L' at the end. long x = 100L
  • A quick IntelliJ tip: if you want to mass replace some text select the portion and then Ctrl + R. Then you will get the menu to replace.
  • Casting means to treat or convert a number from one type to another. byte x = (byte)(12/3)
  • In general Java automatically decides between long and int so you don't have to worry about that.
  • Float and double are used to represent decimal numbers.Java assumes decimal number to be double by default so you might get an error if you use float and double both in your code. Then remember to type cast your variable to be a float.
  • As is the case for int and float same is true for float and double, i.e. 5.0f is not equal to 5.0d
  • Although double takes up twice the space as float but it is advisable to use double variable because modern computers are build on chip level to work effectively with doubles. So it is much faster and efficient method to use double
  • Remember that both float and double are not good for calculation of high precision, BigDecimal is recommended in those scenarios
  • Char is similar to String but you can only use one character. Char takes up 16 bits. Char are stored in Unicode standard.
  • To print/store unicode character look up to the online unicode table and find out the index for your character and then define as following char uni = "\uXXXX", here XXXX is the unicode index.
  • Boolean is a very useful primitive type they are used in pretty much any complex logic structure. They take value true or false.
  • Most common are long,int and double apart from boolean. It is advisable to not use floats or shorts.
  • String is not a primitive type but it enjoys almost all privielege of a primitive type because it is very user friendly.
  • String can be appended using "+" characters. So note that if your variable is a string and you add two of them (say 45.6 +20 ) you'll end up getting 45.620 which is probably not what you were expecting. You need to cast it to double/int.
  • If you add a number to a string you will end up getting a string. So be very careful about the String. Even one of them can mess up your calculations
  • Strings are immutable when you append/edit a string a new string is actually created. A caution is manipulating strings directly is not advisable. Better is to use StringBuffer.

Operators:

  • An operand is a term used to descrbe any object thaat is manipulated by an operator. For example in the statement int myVar = 15+2; + and = are two operators and 15 and 12 are the operands
  • An expression is formmed by combining variables,literals,method return values and operators
  • Comments are ignored by the computer, it is used by the user to get into details of that code.
  • % operator returns remainder. It is called modulus operator.
  • Some of the major abbreviations for operators are,
    • result ++ is equivalent to result = result+1
    • result +=2 is equivalent to result = result +2
    • result -- is equivalent to result = result-1
    • result *= 10 is equivalent to result = result*10
  • If then statement is the most basic of all the control flow statements. It tells your program to execute certain section of code only if a particular test evaluates to true. This is known as conditional logic.
  • Conditional logic uses specific statements in Java to allow us to check a condition and execute certain code based on whether that condition(the expression) is true or false.
  • If statement takes as argument and evaluates t first only then it goes inside the code block. If the statement is evaluated to be false then the computer won't look inside the code block.
  • Even if you are passing through the else , still then put an else statement otherwise it becomes very difficult to track down errors. Also use code blocks when you use if-else.
  • Some Important Notes:
    • "==" checks if LHS , RHS are equal
    • ">=" stands for greater than equal
    • "<=" stands for less than equal
    • ">","<" are greater and lesser respectively
    • "!=" stands for not equals
    • "&&" stands for and
    • "||" stands for or
  • The ternary operator is a shortcut to assigning one of two values to a variable depending on a given condition
  • Example of Ternary Operator: boolean y = x==25?true:false It will check if x=25 then y is set to be true else false.
