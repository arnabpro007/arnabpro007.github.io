Linux Basic Commands

Are you new to Ubuntu just like me? Well, then you have arrived at the perfect place. I will keep adding new Linux hacks and codes in this post. I wish that it will help us to create a better coding community.

Basic Codes:

First Start your linux computer (if you dont have linux but want to install ubuntu check this link: Ubuntu Installation Guide) Then click Ctrl+Alt+T to open a terminal. Terminal is in a way your chemistry lab in ubuntu. All of your experimentations will be executed here. After opening the terminal let's say you are interested to know which directory you are currently in. Then write pwd and press enter. Below you can see my results when I exectured pwd. 

        
            arnab@arnab-HP-Laptop-15-da0xxx:~$ pwd
            /home/arnab
Next let's get into a bit interesting commands.

Sudo:

Sudo means Super User do. It performs as the administrator and runs a chunk of code. Remember by itself sudo can not run anything.

a. apt-get
apt-get is one of the most important Ubuntu commands every beginner must know. It is used to install, update, upgrade and remove any package. apt-get basically works on a database of available packages. Here is the list of different apt-get commands:
1. sudo apt-get update: It tells the system if new updates are available. Note that it will ask for the password of the machine. Remember your exact password. It is extremely important.
